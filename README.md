# Blog site on react
## To start project on localhost:
  - Clone this repository to your PC
  - `npm install` / `yarn install` to install dependencies
  - `npm start` / `yarn start` to start project on localhost
## Technologies used:
  - React
  - Redux
  - SCSS
  - React Router
  - Draft JS
